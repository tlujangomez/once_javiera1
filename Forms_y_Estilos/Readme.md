Trabajo: Creación de un Formulario de Contacto
Introducción

En este trabajo realicé un formulario de contacto utilizando HTML y CSS. La idea principal no era solo que funcionara, sino que también se viera moderno y llamativo, usando colores oscuros y un estilo tipo neón en tonos morados.

¿Cómo desarrollé el trabajo?
1. Creación de la estructura (HTML)

Primero empecé creando el archivo HTML, que es donde va toda la estructura de la página.

Allí hice un formulario donde agregué varios campos importantes como:

Nombre completo
Correo electrónico
Tipo de consulta
Mensaje

También agregué botones para enviar la información o borrar los datos.

Para organizar mejor todo, utilicé div con clases, ya que esto ayuda a que después en CSS sea más fácil aplicar estilos a cada parte.

2. Separación de archivos

Para trabajar de forma más ordenada, utilicé dos archivos:

index.html → donde está toda la estructura
estilo.css → donde está todo el diseño

Luego conecté el CSS con el HTML usando una línea en el head, para que los estilos se aplicaran correctamente.

3. Diseño de la página (CSS)

Después de tener la estructura, empecé a diseñar la página con CSS.

Primero definí algunos colores principales, especialmente:

Morado (color principal)
Negro y tonos oscuros (fondo)
Colores claros para el texto

Esto lo hice usando variables para que fuera más fácil cambiar los colores después si era necesario.

4. Uso del modo oscuro

Decidí hacer la página en modo oscuro porque se ve más moderno.

El fondo de toda la página es oscuro, mientras que el formulario tiene un tono un poco diferente para que se destaque.

5. Efecto neón

Para que el diseño fuera más llamativo, agregué un efecto tipo neón.

Esto lo logré usando sombras en CSS (box-shadow), especialmente en:

Los campos donde se escribe (inputs)
Los botones

Así, cuando el usuario interactúa, los elementos se iluminan como si fueran neón.

6. Interacciones con el usuario

También agregué algunos efectos cuando el usuario usa el formulario:

Cuando se hace clic en un campo, este se ilumina
Cuando se pasa el mouse por los botones, cambian un poco

Esto hace que la página no se vea estática, sino más dinámica.

7. Adaptación a diferentes pantallas

Intenté que el formulario también se vea bien en celulares.

Para esto utilicé una pequeña configuración que hace que los campos se acomoden dependiendo del tamaño de la pantalla.

Resultado final

El resultado fue un formulario que:

Funciona correctamente
Tiene un diseño moderno
Es más atractivo visualmente que uno básico
Se adapta a diferentes dispositivos
Conclusión

En este trabajo aprendí a combinar HTML y CSS para crear una página más completa.

También entendí que no solo importa que algo funcione, sino que también se vea bien, ya que el diseño influye mucho en la experiencia del usuario.
